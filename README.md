# AARC
**先进型抽象线路图画布**  
Advanced Abstract Route Canvas  

## 主要功能
- 绘制线条角度为45度整数倍的线路图，以及车站、水体、出站换乘、线路列表等内容
- 完全支持在所有类型设备（PC/平板/手机）上进行编辑，可双指或滚轮缩放视角

## 特色功能
- 多人同时协作编辑同一张画布
- 可自由插入图片（例如该线路图所属公司logo等）
- 智能的网格/临近车站吸附
- 导出各线路所经站点列表（Excel等格式）

## 架构
- 前端：vue3 + vite + ts
- 后端：aspNetCore
- 双向即时通讯：signalR
- ORM：entityFrameworkCore
